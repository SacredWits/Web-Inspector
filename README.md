# Web-Inspector
Inspect any web application on your android device.
1. View page source
2. View elements
3. View JavaScript errors
4. Execute custom JavaScript code in console
5. View css
6. View Network requests
7. Change viewport size to any custom device manually
8. Change user agent
9. and lot more...
